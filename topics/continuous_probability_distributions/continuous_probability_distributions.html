Continuous Probability distirbutions

layout(matrix(1:3, 1, 3))

col = "darkgreen"

u = seq(0,1,0.1)
plot(u, dunif(u), 
     type = "l",
     xlab = "Test statistic", 
     ylab="density", 
     col = col, 
     main="Uniform distributions" )

polygon(c(u, rev(u) ), c(u*0, dunif(u) ) , col = col) 


T = seq(-4,4,.01)
n  = 5
df = n - 1
col = "red"

plot(T,  dt(T, df), 
     type = "l", 
     xlim = c(-4,4), 
     ylim = c(0,.45), 
     xlab = "T", 
     ylab="density", 
     col = col, 
     main="T-distributions" )

polygon(c(T, rev(T) ), c(T*0, dt(T, df) ) , col = col) 
 

F = seq(0,10,.01)
df.1 = 3
df.2 = 34
col = "darkorange"

plot(F,  df(F, df.1, df.2), 
     type = "l", 
     xlim = c(0,10), 
     xlab = "F", 
     ylab="density", 
     col = col, 
     main="F-distributions" )

polygon(c(F, rev(F) ), c(df(F, df.1, df.2), F*0 ) , col = col)

For all continuous probability distributions

  • Total area is always 1
  • The probability of one specific test statistic is 0
  • x-axis represents the test statistic
  • y-axis represents the probability density
